Input Form

This is a form that we'll use to demonstrate input reactivity. Each input is tied to a display value that reflects what you put into the form. If there is nothing in the form, nothing is displayed. Fancy, no? Let's have a go!

You may have noticed, if you've examined the app.js file carefully, that I've applied the listCtrl Controller to this view. That's perfectly legal. If there were a lot of crossover between the two of them, I might see a lot of benefit in that. Just be careful - if you look in the console, you'll see that the List data was loaded from the backend. You don't want to overload your wire with too many unused API calls!
